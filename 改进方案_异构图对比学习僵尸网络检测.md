# 异构图对比学习僵尸网络检测改进方案

## 一、论文核心方法总结

### 1.1 Bot-DM: 双模态僵尸网络检测方法

**核心思想：**
- 将原始流量字节处理成两种模态进行联合学习
- **Botflow-token模态**：使用Bi-Gram编码payload，通过Transformer学习隐式语义关系
- **Botflow-image模态**：将流量转换为灰度图像，使用CNN+BiLSTM提取空间-时序特征
- 通过**最大化互信息**融合两种模态

**关键技术：**
- Token Embedding + Position Embedding + Segment Embedding
- 多层双向Transformer编码器
- 互信息最大化目标函数

### 1.2 TRUSTED: 概念漂移感知的无监督检测

**核心思想：**
- 三阶段系统：预处理 → 概念漂移识别 → 僵尸网络检测
- 使用**D3N（非线性判别漂移检测器）**检测数据分布变化
- 使用**层次凝聚聚类(HAC)**进行无监督检测

**关键技术：**
- 滑动窗口 + 概念漂移检测
- 判别分类器判断新旧数据分布差异
- 相似度分析识别僵尸网络集群

## 二、当前实现分析

### 2.1 现有架构
```
数据加载 → 异构图构建 → 特征提取(64维) → GAT+GCN编码器 → 分类器
```

### 2.2 现有问题
1. **特征单一**：仅使用统计特征，缺乏语义层面的表达
2. **静态学习**：未考虑数据分布的动态变化（概念漂移）
3. **对比学习不完整**：缺少有效的对比损失设计
4. **泛化能力有限**：对未知僵尸网络的检测效果有待提升

### 3.2 核心改进模块

#### 改进1: 三模态特征融合编码器

**设计思路：** 借鉴Bot-DM的双模态思想，扩展为三模态以更好地利用异构图的特性。

```python
class TriModalFeatureEncoder(nn.Module):
    """
    三模态特征融合编码器
    
    模态1: 统计特征 (Statistical Features)
    - 使用现有的64维统计特征
    - GAT编码器捕捉邻居节点的统计关系
    
    模态2: 语义特征 (Semantic Features)  
    - 将IP节点的通信模式编码为语义序列
    - Transformer编码器学习序列中的隐式语义
    
    模态3: 结构特征 (Structural Features)
    - 图结构信息（度、中心性、社区结构）
    - GCN编码器学习结构表示
    """
    def __init__(self, stat_dim, semantic_dim, struct_dim, hidden_dim):
        super().__init__()
        
        # 模态1: 统计特征编码器
        self.stat_encoder = GATEncoder(stat_dim, hidden_dim)
        
        # 模态2: 语义特征编码器 (Transformer)
        self.semantic_encoder = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(d_model=semantic_dim, nhead=4),
            num_layers=3
        )
        
        # 模态3: 结构特征编码器
        self.struct_encoder = GCNEncoder(struct_dim, hidden_dim)
        
        # 跨模态注意力融合
        self.cross_modal_attention = CrossModalAttention(hidden_dim, num_modalities=3)
        
    def forward(self, stat_x, semantic_x, struct_x, edge_index):
        # 编码各模态
        h_stat = self.stat_encoder(stat_x, edge_index)
        h_sem = self.semantic_encoder(semantic_x)
        h_struct = self.struct_encoder(struct_x, edge_index)
        
        # 跨模态融合
        h_fused = self.cross_modal_attention([h_stat, h_sem, h_struct])
        
        return h_fused, [h_stat, h_sem, h_struct]
```

#### 改进2: 语义特征提取器

**设计思路：** 借鉴Bot-DM的Token思想，但创新性地应用于图节点级别的语义编码。

```python
class NodeSemanticFeatureExtractor:
    """
    节点语义特征提取器
    
    核心创新：
    1. 将每个IP节点的通信序列视为"句子"
    2. 使用Bi-Gram编码目标端口序列和协议序列
    3. 学习节点行为的语义表示
    """
    
    @staticmethod
    def extract_semantic_features(df: pd.DataFrame, node_map: dict) -> torch.Tensor:
        num_nodes = len(node_map)
        semantic_dim = 32
        
        # 1. 构建节点行为序列
        node_sequences = {}
        for ip, group in df.groupby('src_ip'):
            # 按时间排序的行为序列
            sorted_group = group.sort_values('start_time')
            port_seq = sorted_group['dst_port'].values
            proto_seq = sorted_group['protocol'].values
            byte_seq = sorted_group['bytes'].values
            node_sequences[ip] = {
                'ports': port_seq,
                'protos': proto_seq,
                'bytes': byte_seq
            }
        
        # 2. Bi-Gram编码（借鉴Bot-DM）
        port_vocab = {}
        vocab_idx = 0
        features = np.zeros((num_nodes, semantic_dim), dtype=np.float32)
        
        for ip, idx in node_map.items():
            if ip not in node_sequences:
                continue
                
            seq = node_sequences[ip]
            ports = seq['ports']
            protos = seq['protos']
            bytes_vals = seq['bytes']
            
            # 端口Bi-Gram频率
            port_bigram_freq = {}
            for i in range(len(ports) - 1):
                bigram = (int(ports[i]), int(ports[i+1]))
                port_bigram_freq[bigram] = port_bigram_freq.get(bigram, 0) + 1
            
            # 协议分布
            proto_dist = {}
            for p in protos:
                proto_dist[p] = proto_dist.get(p, 0) + 1
            
            # 字节序列统计
            byte_stats = [np.mean(bytes_vals), np.std(bytes_vals), 
                         np.max(bytes_vals), np.min(bytes_vals)]
            
            # 构建语义特征向量
            # [0-15]: 端口Bi-Gram熵（按端口范围分组）
            port_ranges = [(0, 1023), (1024, 49151), (49152, 65535)]
            for i, (low, high) in enumerate(port_ranges):
                count = sum(1 for p in ports if low <= int(p) <= high)
                features[idx, i] = count / (len(ports) + 1e-6)
            
            # [16-19]: 协议语义
            for i, p in enumerate(['tcp', 'udp', 'icmp', 'other']):
                features[idx, 16+i] = proto_dist.get(p, 0) / (len(protos) + 1e-6)
            
            # [20-23]: 字节序列语义
            features[idx, 20:24] = byte_stats
            
            # [24-31]: 端口转换模式熵
            transitions = {}
            for i in range(len(ports) - 1):
                trans_type = classify_port_transition(ports[i], ports[i+1])
                transitions[trans_type] = transitions.get(trans_type, 0) + 1
            
            for i, t in enumerate(['low_to_high', 'high_to_low', 'low_to_low', 'high_to_high', 
                                   'same', 'diverse', 'burst', 'sparse']):
                features[idx, 24+i] = transitions.get(t, 0) / (len(ports) + 1e-6)
        
        return torch.from_numpy(features)
```

#### 改进3: 概念漂移感知的对比学习

**设计思路：** 借鉴TRUSTED的概念漂移检测思想，创新性地融入对比学习框架。

```python
class ConceptDriftAwareContrastiveLearning(nn.Module):
    """
    概念漂移感知的对比学习模块
    
    核心创新：
    1. 使用判别式漂移检测器监控特征分布变化
    2. 动态调整样本权重以适应分布变化
    3. 设计多视图对比损失增强泛化能力
    """
    
    def __init__(self, hidden_dim, num_views=3, drift_threshold=0.7):
        super().__init__()
        self.drift_threshold = drift_threshold
        self.num_views = num_views
        
        # 投影头
        self.projection_head = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 128)
        )
        
        # 漂移检测器 (简化版D3N)
        self.drift_detector = DriftDetector(input_dim=hidden_dim, threshold=drift_threshold)
        
        # 历史特征缓存
        self.register_buffer('feature_history', None)
        self.history_size = 1000
        
    def detect_drift(self, current_features):
        """检测概念漂移"""
        if self.feature_history is None:
            self.feature_history = current_features[:self.history_size]
            return 0.0, None
        
        # 计算新旧数据分布的可区分性
        old_features = self.feature_history
        new_features = current_features[:self.history_size]
        
        # 使用MMD距离衡量分布差异
        mmd_distance = self._compute_mmd(old_features, new_features)
        
        # 更新历史
        self.feature_history = torch.cat([
            new_features,
            self.feature_history[:self.history_size - len(new_features)]
        ], dim=0) if len(new_features) < self.history_size else new_features
        
        return mmd_distance, old_features
    
    def _compute_mmd(self, x, y):
        """计算最大均值差异(MMD)"""
        xx = torch.mm(x, x.t())
        yy = torch.mm(y, y.t())
        xy = torch.mm(x, y.t())
        
        rx = xx.diag().unsqueeze(0).expand_as(xx)
        ry = yy.diag().unsqueeze(0).expand_as(yy)
        
        Kxx = torch.exp(-0.5 * (rx.t() + rx - 2*xx))
        Kyy = torch.exp(-0.5 * (ry.t() + ry - 2*yy))
        Kxy = torch.exp(-0.5 * (rx.t() + ry - 2*xy))
        
        mmd = Kxx.mean() + Kyy.mean() - 2*Kxy.mean()
        return mmd.item()
    
    def compute_contrastive_loss(self, h_views, labels, drift_score):
        """
        计算漂移感知的对比损失
        
        Args:
            h_views: 多视图特征列表 [h_stat, h_sem, h_struct]
            labels: 节点标签
            drift_score: 概念漂移强度
        """
        # 投影到对比空间
        z_views = [self.projection_head(h) for h in h_views]
        
        # 归一化
        z_views = [F.normalize(z, dim=1) for z in z_views]
        
        # 计算多视图对比损失
        total_loss = 0
        
        for i in range(self.num_views):
            for j in range(i+1, self.num_views):
                # 视图间对比
                loss_ij = self._info_nce_loss(z_views[i], z_views[j], labels)
                total_loss += loss_ij
        
        # 漂移自适应权重
        # 当检测到较大漂移时，增加对比损失的权重
        adaptive_weight = 1.0 + drift_score
        
        return total_loss * adaptive_weight / (self.num_views * (self.num_views - 1) / 2)
    
    def _info_nce_loss(self, z1, z2, labels):
        """InfoNCE对比损失"""
        batch_size = z1.size(0)
        
        # 正样本：同一节点的不同视图
        pos_sim = (z1 * z2).sum(dim=1) / 0.1  # temperature
        
        # 负样本：不同节点
        neg_sim = torch.mm(z1, z2.t()) / 0.1
        
        # 构建标签
        labels_mask = labels.unsqueeze(1) == labels.unsqueeze(0)
        
        # InfoNCE损失
        logits = torch.cat([pos_sim.unsqueeze(1), neg_sim], dim=1)
        targets = torch.zeros(batch_size, dtype=torch.long, device=z1.device)
        
        loss = F.cross_entropy(logits, targets)
        return loss


class DriftDetector(nn.Module):
    """判别式漂移检测器 (简化版D3N)"""
    
    def __init__(self, input_dim, threshold=0.7):
        super().__init__()
        self.threshold = threshold
        self.discriminator = nn.Sequential(
            nn.Linear(input_dim, input_dim // 2),
            nn.ReLU(),
            nn.Linear(input_dim // 2, 1),
            nn.Sigmoid()
        )
        
    def forward(self, old_features, new_features):
        """
        检测新旧数据分布是否存在显著差异
        
        Returns:
            drift_score: 0-1之间的漂移强度
        """
        # 标记旧数据为0，新数据为1
        old_labels = torch.zeros(old_features.size(0), 1)
        new_labels = torch.ones(new_features.size(0), 1)
        
        # 合并数据
        all_features = torch.cat([old_features, new_features], dim=0)
        all_labels = torch.cat([old_labels, new_labels], dim=0)
        
        # 判别器尝试区分新旧数据
        with torch.no_grad():
            preds = self.discriminator(all_features)
            
        # 计算AUC作为漂移分数
        # 如果AUC高，说明新旧数据可区分，存在漂移
        from sklearn.metrics import roc_auc_score
        drift_score = roc_auc_score(all_labels.cpu().numpy(), preds.cpu().numpy())
        
        return drift_score
```

#### 改进4: 动态数据增强策略

```python
class DynamicGraphAugmentor:
    """
    动态图数据增强器
    
    策略：
    1. 节点丢弃：根据节点重要性自适应丢弃
    2. 边扰动：保持语义结构的边修改
    3. 特征掩码：基于特征重要性的掩码
    """
    
    def __init__(self, aug_ratio=0.2, adaptive=True):
        self.aug_ratio = aug_ratio
        self.adaptive = adaptive
        
    def augment(self, data, node_importance=None):
        """生成增强视图"""
        aug_data = data.clone()
        
        # 1. 自适应节点丢弃
        if self.adaptive and node_importance is not None:
            # 保留重要节点，丢弃不重要的
            keep_prob = 1 - self.aug_ratio * (1 - node_importance)
            keep_mask = torch.rand(data.num_nodes) < keep_prob
        else:
            keep_mask = torch.rand(data.num_nodes) > self.aug_ratio
            
        # 2. 边扰动
        edge_index = aug_data['ip', 'flow', 'ip'].edge_index
        num_edges = edge_index.size(1)
        edge_mask = torch.rand(num_edges) > self.aug_ratio / 2
        aug_data['ip', 'flow', 'ip'].edge_index = edge_index[:, edge_mask]
        
        # 3. 特征掩码
        x = aug_data['ip'].x
        feat_mask = torch.rand(x.size(1)) > self.aug_ratio
        aug_data['ip'].x = x * feat_mask.float()
        
        return aug_data
```

### 3.3 完整的改进模型架构

```python
class ImprovedBotnetDetector(nn.Module):
    """
    改进的僵尸网络检测器
    
    整合：
    1. 三模态特征融合
    2. 概念漂移感知的对比学习
    3. 动态数据增强
    """
    
    def __init__(self, stat_dim=64, semantic_dim=32, struct_dim=16, hidden_dim=128):
        super().__init__()
        
        # 特征编码器
        self.tri_modal_encoder = TriModalFeatureEncoder(
            stat_dim, semantic_dim, struct_dim, hidden_dim
        )
        
        # 对比学习模块
        self.contrastive_module = ConceptDriftAwareContrastiveLearning(
            hidden_dim, num_views=3
        )
        
        # 数据增强器
        self.augmentor = DynamicGraphAugmentor(aug_ratio=0.2)
        
        # 分类器
        self.classifier = nn.Sequential(
            nn.Linear(hidden_dim, 64),
            nn.PReLU(),
            nn.Dropout(0.3),
            nn.Linear(64, 32),
            nn.PReLU(),
            nn.Linear(32, 1)
        )
        
        # 节点重要性估计器（用于自适应增强）
        self.importance_estimator = nn.Sequential(
            nn.Linear(hidden_dim, 1),
            nn.Sigmoid()
        )
        
    def forward(self, data, return_all=False):
        # 提取三种模态特征
        stat_x = data['ip'].stat_x  # 统计特征
        semantic_x = data['ip'].semantic_x  # 语义特征
        struct_x = data['ip'].struct_x  # 结构特征
        edge_index = data['ip', 'flow', 'ip'].edge_index
        
        # 三模态编码
        h_fused, h_views = self.tri_modal_encoder(
            stat_x, semantic_x, struct_x, edge_index
        )
        
        # 分类
        logits = self.classifier(h_fused)
        
        if return_all:
            return logits, h_fused, h_views
        return logits
    
    def compute_loss(self, data, labels, old_features=None):
        """计算总损失"""
        # 前向传播
        logits, h_fused, h_views = self.forward(data, return_all=True)
        
        # 分类损失
        cls_loss = F.binary_cross_entropy_with_logits(
            logits.squeeze(), labels.float()
        )
        
        # 概念漂移检测
        drift_score, _ = self.contrastive_module.detect_drift(h_fused.detach())
        
        # 对比学习损失
        con_loss = self.contrastive_module.compute_contrastive_loss(
            h_views, labels, drift_score
        )
        
        # 总损失
        total_loss = cls_loss + 0.5 * con_loss
        
        return total_loss, {
            'cls_loss': cls_loss.item(),
            'con_loss': con_loss.item(),
            'drift_score': drift_score
        }
```
