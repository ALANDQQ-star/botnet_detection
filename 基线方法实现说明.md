# Bot-AHGCN 基线方法实现说明

## 概述

根据文献一《Multi-attributed heterogeneous graph convolutional network for bot detection》（Information Sciences 537 (2020) 380–393）中提出的 Bot-AHGCN 方法，本系统已实现并集成了该基线方法。

## 实现内容

### 1. 核心文件：`bot_ahgcn_baseline.py`

实现了以下关键组件：

#### 1.1 BotAHGCNGraphBuilder（异构信息网络构建器）
- **功能**：构建属性异构信息网络（AHIN）
- **节点类型**：
  - IP节点（源IP和目标IP）
  - Port节点（端口）
  - Protocol节点（协议）
  - Request节点（请求）
  - Response节点（响应）
- **关系类型**（对应论文中的R1-R10）：
  - R1: Source IP → Destination IP（连接关系）
  - R2, R7: IP → Protocol（使用关系）
  - R3, R8: IP → Port（使用关系）
  - R4, R9: IP → Request（发送/接收关系）
  - R5, R10: IP → Response（接收/发送关系）
  - R6: Protocol → Port（利用关系）

#### 1.2 MetaPathSimilarity（元路径相似性计算器）
- **功能**：基于元路径和元图计算IP节点之间的相似性
- **元路径类型**：
  - P1: 使用相同协议的主机
  - P2: 发送相同请求的主机
  - P3: 访问相同端口的主机
  - P8: 向相同目标发送相同请求的主机
- **元图类型**：
  - M1: 同时使用相同端口和发送相同请求的主机
  - M3: 同时使用相同端口和协议的主机
- **实现公式**：
  - 公式(1): SM(hi, hj) = Σm wm * 2*|{hi→j ∈ Pm}| / (|{hi→i ∈ Pm}| + |{hj→j ∈ Pm}|)
  - 公式(2): SG(hi, hj) = Σm wg * 2*CouMG(hi, hj) / (CouMG(hi, hi) + CouMG(hj, hj))

#### 1.3 BotAHGCNModel（Bot-AHGCN模型）
- **架构**：
  - 两个独立的GCN分支：一个用于元路径相似性（AM），一个用于元图相似性（AG）
  - 可学习系数α用于融合两个分支（公式7）
  - 分类器层用于最终预测
- **输入**：
  - x: 节点特征矩阵 (N × 100)
  - AM: 基于元路径的相似性矩阵 (N × N)
  - AG: 基于元图的相似性矩阵 (N × N)
- **输出**：
  - 节点分类logits (N × 1)

#### 1.4 BotAHGCNTrainer（训练器）
- **功能**：训练Bot-AHGCN模型
- **损失函数**：BCEWithLogitsLoss
- **优化器**：Adam
- **特征**：
  - 支持训练和推理
  - 模型保存和加载

  - 加载基线模型
  - 执行威胁检测
  - 生成可视化结果

## 使用方法

### 方式1：通过Web界面

1. 打开应用首页
2. 在"检测方法"下拉框中选择：

   - "基线方法 (Bot-AHGCN)" - 使用文献一的方法

## 方法对比

| 特性 | 现有方法 (GAT+GCN) | 基线方法 (Bot-AHGCN) |
|------|---------------------|------------------------|
| **图类型** | 同构图（仅IP节点） | 异构图（IP、Port、Protocol、Request、Response） |
| **特征维度** | 64维统计特征 | 100维PCA特征 |
| **模型架构** | GAT + GCN | 双分支GCN（元路径+元图） |
| **相似性建模** | 基于边权重 | 基于元路径和元图 |
| **论文依据** | - | Information Sciences 537 (2020) |
| **理论优势** | 简单高效 | 更丰富的语义信息，更好的可解释性 |

## 技术细节

### 数据预处理
1. **端口映射**：保留前5000个最常见端口
2. **请求映射**：保留前1000个最常见请求
3. **响应映射**：保留前1000个最常见响应
4. **特征提取**：使用PCA将统计特征投影到100维

### 相似性计算
1. **元路径相似性**：通过中间节点类型计算共现矩阵
2. **元图相似性**：使用Hadamard积组合多个元路径
3. **权重学习**：使用预定义权重（可根据数据调整）

### 模型训练
- **学习率**：0.01
- **批次大小**：全图训练（无采样）
- **训练轮数**：5轮（可配置）
- **Dropout**：0.3
- **激活函数**：PReLU

## 预期性能

根据论文报告（CTU-13数据集）：
- **精确率（Precision）**：98.24%
- **召回率（Recall）**：98.31%
- **Micro-F1**：98.27%

实际性能可能因数据集和超参数设置而有所不同。

## 注意事项

1. **内存占用**：Bot-AHGCN需要构建完整的相似性矩阵，对于大规模网络可能占用较多内存
2. **计算时间**：相似性计算是计算密集型操作，可能需要较长时间
3. **模型文件**：基线方法会生成独立的模型文件 `bot_ahgcn_baseline.pth`
4. **兼容性**：两种方法生成的可视化结果格式相同，可以无缝切换
